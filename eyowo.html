Overview

Project

Eyowo is a Nigerian company solving a payment issue in the country by providing a platform where payments can be made online or offline to everyone. Built for users with or without banking credentials.

Responsibilities

Building a fast and responsive marketing Web site for the company as well as develop the interface for the Web app of the product. Worked as part of the design team ot ensure that users with varying devices get an equally rich platform experience.

Role

Web Developer, UI Developer

Project Duration

4 - 6 Months
